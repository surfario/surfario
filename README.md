- **[CRIT — the Conversion Readiness Inspection Toolkit](https://github.com/surfario/crit)** - a framework-driven prompt series (Shaan Puri's click test, Hormozi's value equation, Zig Ziglar's objections + three adversarial personas) that tears a landing page apart and returns a prioritized fix list. Teaching ~200+ founders at Oneday, I found that while there's no shortage of systems, the landing page itself is an art that's hard to master — so I released this to help more founders get better at it, faster. Quick (~10 min) and full paths, plus a post-click funnel audit.
